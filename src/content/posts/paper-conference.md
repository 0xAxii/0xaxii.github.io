## 주요 보안 컨퍼런스

보안 분야의 4대 탑 컨퍼런스를 소개합니다.

### IEEE S&P (Oakland)
- 시스템 보안, 네트워크 보안 분야의 최고 학회
- 매년 5월 개최

### USENIX Security
- 실용적인 보안 연구 중심
- 매년 8월 개최

### ACM CCS
- 컴퓨터 보안 분야 전반
- 매년 11월 개최

### NDSS
- 네트워크 및 분산 시스템 보안
- 매년 2~3월 개최

## 2026년 주목할 연구 분야

### 1. AI/ML 보안
- LLM(대규모 언어 모델) 보안 위협
- Adversarial Attack 방어 기법
- AI 기반 취약점 자동 탐지

### 2. 공급망 보안 (Supply Chain Security)
- 오픈소스 소프트웨어 공급망 공격
- SBOM(Software Bill of Materials) 활용
- 의존성 혼동 공격 방어

### 3. 하드웨어 보안
- 사이드 채널 공격 연구
- TEE(Trusted Execution Environment) 취약점
- RISC-V 보안 확장

### 4. 클라우드 네이티브 보안
- 컨테이너/쿠버네티스 보안
- 서버리스 환경 보안
- 마이크로서비스 간 통신 보안

## 논문 읽기 팁

1. **Abstract와 Introduction**을 먼저 읽고 전체 맥락을 파악합니다
2. **Threat Model**을 통해 공격자 모델을 이해합니다
3. **Evaluation** 섹션에서 실험 결과를 확인합니다
4. **Related Work**에서 관련 연구를 추적합니다
